# Domain-Driven Design (DDD) Project

This project implements a sample application based on the DDD architectural pattern, utilizing strategic and tactical patterns.

## Features
- **Bounded Context**: Clear boundaries established.
- **Aggregates and Entities**: Deep domain modeling.
- **Repositories**: Standard patterns to manage aggregates.
- **Value Objects**: Defining unidentifiable characteristics.

## Technologies Used
- C# 12
- .NET 8
- Entity Framework Core
- xUnit & Moq
